DaeSoop - 익명의 대나무숲

계정 및 데이터 삭제 방법

DaeSoop은 익명 서비스로 별도의 회원가입이 없습니다.

아래 방법으로 모든 데이터를 완전히 삭제할 수 있습니다.

방법 1: 앱 삭제

앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.

Android

  1. 설정 앱을 열고 앱 또는 애플리케이션을 선택합니다.
  2. DaeSoop을 찾아 선택합니다.
  3. 삭제 또는 제거를 누릅니다.

iOS

  1. 홈 화면에서 DaeSoop 앱 아이콘을 길게 누릅니다.
  2. 앱 제거를 선택합니다.
  3. 앱 삭제를 눌러 확인합니다.

방법 2: 앱 데이터 초기화

앱을 삭제하지 않고 데이터만 삭제하려면 다음 방법을 이용하세요.

Android

  1. 설정 앱을 열고 앱 또는 애플리케이션을 선택합니다.
  2. DaeSoop을 찾아 선택합니다.
  3. 저장공간을 선택합니다.
  4. 데이터 삭제 또는 캐시 삭제를 누릅니다.

iOS

  1. 설정 앱을 열고 일반 > iPhone 저장 공간을 선택합니다.
  2. DaeSoop을 찾아 선택합니다.
  3. 앱 정리하기를 눌러 데이터를 삭제합니다.

소셜 로그인 계정 삭제

소셜 로그인(Google, Apple, 카카오 등)으로 가입한 경우 아래 방법으로 계정을 삭제할 수 있습니다.

앱 내 계정 삭제

  1. DaeSoop 앱을 실행합니다.
  2. 설정 (⚙️) 메뉴로 이동합니다.
  3. 계정 관리를 선택합니다.
  4. 계정 삭제를 누르고 확인합니다.

주의: 계정을 삭제하면 작성한 게시물, 댓글, 반응 기록 등 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다.

삭제되는 데이터

계정 또는 앱을 삭제하면 다음 데이터가 모두 삭제됩니다:

  • 익명 ID 및 프로필 정보
  • 작성한 모든 게시물
  • 작성한 모든 댓글
  • 반응(좋아요, 공감 등) 기록
  • 알림 설정 및 앱 설정
  • 소셜 로그인 연동 정보 (해당되는 경우)
